# Vue Showcase Monorepo

A collection of Vue 3 and Nuxt applications showcasing modern web design and real-time backend integration.

## Projects Included

- **Main Landing Page**: Central hub for navigating the ecosystem.
- **Airbnb Clone**: A website to book houses for trips.
- **Ecommerce App**: An online shop to buy clothes.
- **LMS Platform**: A school website for learning online.
- **Portfolio App**: A simple and clean website to show my work and skills.

## Tech Stack

- **Frontend**: Vue 3, Nuxt 3, Vite, TypeScript
- **Backend**: Convex (Real-time database and serverless functions)
- **Styling**: Vanilla CSS, Modern Typography
- **Animations**: GSAP, AOS

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run All Applications**:
   This command starts all frontends and their respective Convex backends concurrently.
   ```bash
   npm run dev:all
   ```

3. **Access the Apps**:
   - Landing Page: [http://localhost:5174](http://localhost:5174)
   - LMS Platform: [http://localhost:3001](http://localhost:3001)
   - Airbnb Clone: [http://localhost:3002](http://localhost:3002)
   - Ecommerce App: [http://localhost:3003](http://localhost:3003)
