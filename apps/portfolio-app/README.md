# Developer Portfolio (Vue 3 + GSAP + AOS)

A simple and clean website to show my work and skills. It features advanced animations and a premium dark-mode aesthetic.

## Features

- **Advanced Animations**: Powered by GSAP for smooth, complex motion sequences.
- **Scroll Effects**: AOS (Animate On Scroll) integration for engaging content reveal.
- **Modern UI**: Clean, responsive design optimized for all devices.
- **State Management**: Pinia for lightweight and efficient state handling.

## Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Development Server**:
   ```bash
   npm run dev -- --port 5174
   ```

## Preview

This app is a static showcase and does not require authentication.
